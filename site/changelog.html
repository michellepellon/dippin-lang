Changelog

All notable changes to dippin-lang. Versions follow semver.

v0.11.0 2026-03-27

Added

  • DIP126 lint rule: subgraph ref: file validation — warns when referenced workflow file does not exist on disk.
  • dippin watch command: file watcher that re-runs lint on .dip changes with 200ms debounce.
  • dippin test --coverage flag: edge coverage summary showing which workflow edges were/weren't traversed by test scenarios.
  • Tree-sitter grammar in editors/tree-sitter-dippin/ — grammar.js, external scanner for indentation, highlight queries. Enables syntax highlighting in Neovim, Helix, and Zed.
  • WASM playground — browser-based editor with live parse, lint, and format via WebAssembly.
  • gemini-3.1-pro-preview-customtools added to model catalog.
  • 35 diagnostic codes total.

Fixed

  • CI failures: errcheck, funlen, and misspell violations resolved.
  • Migration parity: model names in .dip files now match DOT originals.
