> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the regex-based shell command binary extraction (DIP125) with proper AST-based parsing using `mvdan.cc/sh/v3`, eliminating false positives on variable assignments, command substitutions, and other shell constructs.

**Architecture:** The `extractBinary` function in `lint_tool_helpers.go` currently uses regex preamble matching and naive first-token extraction. Replace it with a function that parses the command through `sh/syntax.NewParser`, walks the AST for `CallExpr` nodes, and returns the first non-builtin command name. This inherently handles assignments, pipes, subshells, heredocs, conditionals, etc.

**Tech Stack:** Go, `mvdan.cc/sh/v3` (pure Go shell parser, powers `shfmt`)

```go
import (
	"mvdan.cc/sh/v3/syntax"
	// ... existing imports
)

// extractBinary parses a shell command and returns the first non-builtin
// command name. Uses a proper shell AST parser to correctly handle
// variable assignments, pipes, subshells, command substitution, etc.
func extractBinary(command string) string {
	parser := syntax.NewParser(syntax.KeepComments(false))
	prog, err := parser.Parse(strings.NewReader(command), "")
	if err != nil {
		return "" // unparseable — DIP123 handles syntax errors
	}
	var bin string
	syntax.Walk(prog, func(node syntax.Node) bool {
		if bin != "" {
			return false // already found
		}
		call, ok := node.(*syntax.CallExpr)
		if !ok {
			return true // keep walking
		}
		if len(call.Args) == 0 {
			return true // pure assignment, skip
		}
		name := extractWordLiteral(call.Args[0])
		if name != "" && !isShellBuiltin(name) {
			bin = name
			return false
		}
		return true
	})
	return bin
}

// extractWordLiteral returns the literal string value of a simple Word,
// or "" if the word contains expansions/substitutions.
func extractWordLiteral(w *syntax.Word) string {
	if len(w.Parts) != 1 {
		return ""
	}
	lit, ok := w.Parts[0].(*syntax.Lit)
	if !ok {
		return ""
	}
	return lit.Value
}
```

Delete the old `shellPreamble` regex, old `extractBinary`, and `firstToken` functions.

**Important:** The `syntax.Walk` callback uses a closure to capture `bin`. The walker visits every node in the AST depth-first. `CallExpr` with `Args` is a real command invocation. `CallExpr` without `Args` is a pure assignment. The first non-builtin command name found is returned.

**Note on shell builtins:** `set`, `cd`, `export`, `mkdir` are currently in the `shellPreamble` regex but NOT all in `shellBuiltins`. Check and add `set`, `cd`, `export`, and `mkdir` to `shellBuiltins` if missing:
